MetaMask brings Ethereum to your web browser.

About

MetaMask is a bridge that connects the web of today with the web of tomorrow. It allows you to run Ethereum dapps right in your browser without installing plugins or running an Ethereum node. Found a cool dapp? Send a friend a link of it running directly in MetaMask. Our mission is to make Ethereum as easy to use as possible.

Progress

MetaMask is currently in a proof-of-concept stage. We will be engaging beta-testers in the near future. Sign up below to learn more. MetaMask is made possible by support from Ethereum DEV Grants.
